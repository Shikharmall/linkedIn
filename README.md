# LinkedIn

## Backend
  ### Technologies Used
     Node JS, Express JS, MongoDB Atlas
  ### Run Locally
     npm install
     set environment (DATABASE = "MONGODBATLAS", DATABASEURL="confidential-ask-me", CONFIG_SECRET_KEY ="confidential-ask-me")
     nodemon index.js

## Frontend
  ### Technologies Used
     React JS, Vite , Tailwind CSS 
  ### Run Locally
     npm install
     set environment (VITE_ENV="LOCAL")
     npm run dev
